# Projet-Web-Graphique-et-Design-des-Donn-es

Les données utilisées couvrent les Jeux Olympiques jusqu’à l’édition la plus récente disponible au moment de la réalisation du projet. Elles devront être mises à jour après les Jeux de Milan–Cortina 2026.
Les données ne sont pas intégrées de manière dynamique via une API, afin de garantir la portabilité et la simplicité de remise du projet. Néanmoins, une mise à jour automatique aurait été techniquement possible.
